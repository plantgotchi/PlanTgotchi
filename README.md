# PlanTgotchi
다마고치 컨셉의 야외 식물 케어 서비스입니다. 날씨 데이터를 기반으로 식물의 관수 일정을 자동으로 조정하고, RAG 기반 AI 챗봇을 통해 식물 관련 질문에 맞춤형 답변을 제공합니다.

## 프로젝트 소개
PlanTgotchi는 실제 기상 데이터와 식물 도감 지식을 결합하여, 사용자가 직접 신경 쓰지 않아도 식물이 필요한 케어 타이밍을 알려주는 지능형 식물 관리 서비스입니다. 단순 알림이 아니라, 증발산량(ETo) 연산을 기반으로 한 과학적 근거의 동적 스케줄링을 제공합니다.

## 주요 기능

### 1. 날씨 연동 동적 관수 스케줄러
- OpenWeatherMap API로 실시간 기온·습도·강수량 데이터 수신
- FAO Hargreaves 공식으로 기준증발산량(ETo) 직접 연산
- 폭염·강수 조건에 따라 관수 주기를 자동으로 앞당기거나 미룸
- 모든 조정 기준은 기상청 폭염특보 기준 및 FAO 유효강수량 기준에 근거

### 2. RAG 기반 식물 챗봇
- 식물 도감 데이터를 MiniLM으로 임베딩하여 ChromaDB에 벡터 저장
- 사용자 질문과 가장 유사한 도감 청크 Top-3 검색 (코사인 유사도 기반)
- Gemma 2 2B 모델이 검색된 정보와 실시간 날씨 데이터를 결합해 맞춤형 답변 생성
- 정제된 프롬프트(시스템 지침 + 데이터 주입) 적용으로 환각 최소화

### 3. 통합 케어 캘린더
- 여러 식물의 물주기·영양제 일정을 한 화면에서 확인
- 날씨로 인한 일정 변경 시 변경 사유를 함께 표시

### 4. 게임화된 가상 정원 UI
- 식물 상태를 애니메이션 이미지로 직관적으로 표현

## 기술 스택
- Backend: Django
- AI 모델: Gemma 2 2B (HuggingFace Inference API) 
- 벡터 검색: ChromaDB, Sentence-Transformers (MiniLM)
- 날씨 데이터: OpenWeatherMap API
- 증발산량 연산: FAO Hargreaves 공식 (Penman-Monteith 간이형)
- 식물 데이터: 국가생물종지식정보 API

## 관수 주기 조정 로직
관수 주기 조정 기준은 임의로 설정한 수치가 아니라 다음의 공식 근거를 따릅니다.

- 폭염경보 (35℃ 이상) - 기상청 폭염특보 2단계 기준 | 1일 앞당기기
- 폭염주의보/고온건조 (30~33℃, 습도 40% 이하) - 수분 스트레스 구간 | 당일 12시간 앞당기기 
- 정상 기후 (15~25℃) - 변동 없음
- 약한 강수 (5mm 미만) - 농촌진흥청 — 표면만 적심, 뿌리 미도달 | 변동 없음
- 정식 강수/폭우 (10mm 이상) - FAO 유효강수량 기준(토양 포화) | 1일 미루기 

## RAG Top-K 설정
검색 청크 개수는 K=3으로 설정했습니다. 식물 케어에 필요한 핵심 정보를 1. 생육 환경 2. 관수 주기 3. 병해 주의사항 3가지 카테고리로 정의하고, 각 카테고리당 1개의 청크를 가져오도록 구성했습니다. K가 너무 작으면 정보가 부족하고, 너무 크면 관련 없는 데이터가 포함되어 응답 품질이 저하되기 때문에 3을 최적값으로 선정했습니다.
